# Sistema de Clientes - Python

Projeto simples de gerenciamento de clientes desenvolvido em Python.

Este projeto foi criado para praticar conceitos de backend como:
- API com Flask
- Banco de dados SQLite
- Organização de código
- Uso de Git e GitHub

## Tecnologias utilizadas

- Python
- Flask
- SQLite
- HTML (interface simples)

## Estrutura do projeto

```
sistema_clientes
│
├── app.py
├── database.py
├── services.py
├── utils.py
├── main.py
├── clientes.db
├── templates
│   └── index.html
└── README.md
```

## Funcionalidades

- Criar cliente
- Listar clientes
- Salvar dados no banco SQLite
- Interface web simples

## Como executar o projeto

1. Clone o repositório

```
git clone https://github.com/Alessandro-Prado/sistema-clientes-python.git
```

2. Entre na pasta do projeto

```
cd sistema-clientes-python
```

3. Instale o Flask

```
pip install flask
```

4. Execute o projeto

```
python app.py
```

5. Acesse no navegador

```
http://127.0.0.1:5000
```
